# Leaderboard (convoro-leaderboard)

A first-party Convoro extension that adds a **top-contributors leaderboard**.

Members are ranked by reputation — posts written, topics started, and reactions
received on their posts — on a themed `/leaderboard` page, with a link added to
the header. Install it only if your community wants the friendly competition;
otherwise it stays out of your way.

Reputation = posts + (reactions received × 2) + topics started.
